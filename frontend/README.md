# frontend

## Project setup
```
npm install
```
Create a `.env` file to set the location of your API:
```
VUE_APP_API = "/"
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
